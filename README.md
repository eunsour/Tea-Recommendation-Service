# Tea-Recommendation-Service
성분 데이터를 활용한 맞춤형 차(茶) 추천 서비스 - 차누<br><br>
  
## 기획 배경
**차누**는 ‘왜 카누처럼 대중에게 보급화된 차의 브랜드는 없을까?’ 라는 생각에서 시작된 프로젝트 입니다. 차누는 설문조사를 통해 사용자가 원하는 효능의 차를 추천하고 어떠한 효능과 부작용이 있는지 알 수 있는 서비스로 기획되었습니다.<br><br>
